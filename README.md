## Dotfiles
My portable dev environment (OSx, Linux)

## Install
<pre>
git clone git@github.com:wellsjo/dotfiles.git ~/.dotfiles && source ~/.dotfiles/bash/profile && wells_install
</pre>

## Features
- vim config
- tmux config
- git config
- git auto-completion
- ssh auto-completion
- liquid shell prompt

## Special Commands
SSH auto-install dotfiles
```
sshw <host>
```
Udpate dotfiles and submodules
```
wupdate
```

# Dependencies
- tmux
- exuberant-ctags
